Frequently Asked Questions

Can I import my data from another plant records system like BG-Base, BG-Record, IrisBG or Bauble?

We're working on it. This is a major feature for us that we hope to roll out soon. If this is a deal breaker for you then contact us so we know how if we need to pick up the pace.

Do you support feature X?

Maybe. Contact us and we can talk about it. Maybe we can work something out.

Will Bauble be available in other languages?

Definitely. Hopefully soon. We hope to start with Spanish. If you would like to help with translations please contact us.

Is the original Bauble desktop application still available?

Yes. For now you can still download it at bauble.belizebotanic.org. We're going to be rebranding it as Bauble Classic and make it available for download here as well.

I found a bug. What can I do about it?

You can file a bug at GitHub or just contact us.
